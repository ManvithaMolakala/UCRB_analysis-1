# UCRB_analysis
Exploratory modeling of StateMod hydrology under different experimental designs. This repository contains all of the code to reproduce the analysis in the following paper:

Quinn, J.D., Hadjimichael, A., Reed, P.M., and Steinschneider, S. (2020). Can exploratory modeling of water scarcity vulnerabilities and robustness be scenario neutral? Earth and Space Science Open Archive. doi: 10.1002/essoar.10503258.1.
